# Reverse a String

**Difficulty Level:** Basic

**Problem Statement:**
Given a string `s`, print its reverse.

**Examples:**
Input: hello  
Output: olleh  

Input: GFG  
Output: GFG  

**Expected Time Complexity:** O(n)  
**Expected Auxiliary Space:** O(1)  

**Company Tags:** None  
**Topic Tags:** Strings, Loops, Basic Programming
